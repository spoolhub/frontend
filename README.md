# Spoolhub Frontend

## Setup (TLDR;)

```bash
pnpm install
pnpm run dev
```


## Start local server

### Start development server

```bash
# HTTP
pnpm run dev
```

### Start production build server

```bash
# HTTP
pnpm run build;
pnpm run preview;
```

## Run test

### Unit test

```bash
pnpm test:unit
pnpm test:unit:ui
```

### E2E test

```bash
pnpm test:e2e
pnpm test:e2e:ui
```
